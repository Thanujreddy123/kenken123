KenKen Puzzle Rules

Unleash your inner puzzle master with KenKen!

Grid and Numbers

  • The grid is n x n squares, where n is a positive integer greater than 1 (like the one above).
  • Each square can only contain one number.
  • The numbers used in the puzzle range from 1 to n (inclusive).
  • No number can be repeated in any row or column.

Cages

  • The grid is divided into cages, outlined by heavy lines.
  • Each cage contains a target number and an operation symbol (+, -, *, /).
  • The numbers in a cage must combine to produce the target number using the specified operation.
  • Each number can be used multiple times within a cage, as long as it doesn't violate the row and column constraints.
  • Single-cell cages already have their number filled in and cannot be changed.
